# world-settings
Set your code world.

## 设置powershell配置
1. 安装powershell7
2. 修改powershell的脚本执行策略
   ```bash
   Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope LocalMachine
   ```
4. 执行命令
    ```bash
    cp Microsoft.PowerShell_profile.ps1 $HOME\Documents\PowerShell\Microsoft.PowerShell_profile.ps1
    ```
